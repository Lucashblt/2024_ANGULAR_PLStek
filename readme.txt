---Projet WE4B: site web du PL$tek---

Requis:
- Angular 13
- XAMPP avec PhpMyAdmin
- Python 3 accessible avec la commande 'python3' (pour ajouter un exercice au format markdown étendu)

Setup:
- Placer le répertoire du projet à la racine du répertoire 'htdocs' (l'API doit être accessible avec le chemin "localhost/PLSres/api/...")
- Importer la base de données (dans le répertoire 'db') depuis PhpMyAdmin
- Exécuter la commande 'pip install -r requirements.txt' depuis le répertoire 'api/compiler' (pour pouvoir ajouter des exercices)
- Exécuter la commande 'npm install' depuis la racine du projet pour installer les modules nodes

Lancer:
- Veiller à Apache et MySQL soient lancés depuis XAMMPP
- Lancer la commande 'ng serve' depuis la racine du projet

Voilà ! Si tout se passe bien le projet devrait tourner sur votre machine et vous pouvez vous rendre sur le site avec l'url donné après avoir lancé 'ng serve'
